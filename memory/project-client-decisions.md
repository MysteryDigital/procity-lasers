## Colour palette — agreed 2026-07-02

Confirmed palette (see `spec-design.md §4a` for full token table and contrast ratios):

- `#1c425c` — accent (steel blue)
- `#41413e` — neutral dark (warm charcoal)
- `#d7d7d7` — neutral light
- `#8d8d8d` — mid grey (nudged to `#6b6b6b` for WCAG AA compliance)

Surface colour `#f6f5f2` (warm white) is a derived token, not in the original swatch.

`#0f2d3f` — added as `surface-dark` token (deep navy) to support the homepage design prototype; used for footer background and the scrolled/opaque nav state. Not in the original swatch but present throughout the prototype. Added 2026-07-02.

## Typeface — confirmed 2026-07-02

**DM Sans** is the site typeface, chosen by the client/owner. Apply to all UI text. No other sans-serif faces unless a second face is explicitly approved.
